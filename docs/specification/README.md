# specification — 仕様ドキュメント

マリオ風 2D 横スクロールアクションゲームの仕様書です。コード着手前のドラフトであり、確定事項と未決事項が混在します。各ファイルの末尾に「未決事項」セクションを設けています。

## 構成

| ファイル | 内容 |
| --- | --- |
| [vision.md](vision.md) | プロジェクトの目的・コンセプト・スコープ |
| [gameplay.md](gameplay.md) | ゲーム全体の流れ、勝敗条件、進行ループ |
| [mechanics.md](mechanics.md) | プレイヤー挙動・敵・アイテム・物理ルール |
| [level-design.md](level-design.md) | ステージ構成と難易度カーブの方針 |
| [controls.md](controls.md) | 入力デバイスとキー配置 |
| [assets.md](assets.md) | 必要なグラフィック・サウンドアセットの一覧 |

## 表記ルール

- **【確定】** … 方針として決まっているもの
- **【仮】** … たたき台。後で見直す
- **【未決】** … 要相談
