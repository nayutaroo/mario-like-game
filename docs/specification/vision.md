# vision — プロジェクトの目的とコンセプト

## 目的

マリオ風の 2D 横スクロールアクションゲームを個人制作する。

## コンセプト

- **【確定】** Web ブラウザ上で動作する。インストール不要で URL を開けば遊べる。
- **【確定】** キーボード操作で「走る」「ジャンプ」などの基本アクションが可能。
- **【確定】** 初代スーパーマリオブラザーズ世代の横スクロールアクションを参考にする（あくまで参照、固有名詞は本作に持ち込まない）。
- **【確定】** 「パワーアップ」によってプレイヤーの状態が変化する仕組みを採用する。マリオの直接コピーにならないよう、独自モチーフ（植物・果実系）に置き換える。
- **【確定】** ポップでカートゥーン調のビジュアル。主人公は丸いヒヨコのキャラクター「ピヨ」。
- **【確定】** ピクセルアート（ドット絵）調。1 タイル = 32×32 px を基準とする（実装段階で再調整可）。
- **【確定】** ベース解像度は 16:9 ワイド、1280×720 px。ブラウザ表示はアスペクト比固定でスケールする。
- **【確定】** 初期リリースは**無音**。BGM / SE は将来フェーズで検討する。

## スコープ

- **【確定】** 小規模（個人制作）。1 ワールド・全 5 面（1-1〜1-4 通常面 + 1-5 ボス面）+ 任意のボーナス面で完結する。
- **【確定】** 通常面は基本的に横スクロール。一部ステージで**縦スクロールも許容**する（例: 1-4「空」）。
- **【確定】** **ボス**を最終面に配置する。
- **【確定】** **隠しエリア・ボーナス面**を含める。
- **【確定】** **1UP 専用アイテム**を含める。
- **【確定】** マルチプレイ・オンライン要素は含めない。
- **【確定】** ステージエディタ等のツール群はスコープ外。

## 非目標 (Non-goals)

- 3D 化
- スマホタッチ操作（将来検討としても、初期実装ではキーボードのみ）
- アカウント / セーブデータのクラウド同期
- 課金要素

## ターゲット環境

- **【確定】** モダンブラウザの最新版（Chrome / Firefox / Safari / Edge）。古い IE 等は対象外。
